Tutorial

Setup

Install the following packages with pip.
  • pip install beautifulsoup4
  • pip install schedule
  • pip install pymongo

Each scrapper requires it's own library. They are as follows:
  • pip install python-craigslist
  • pip install meetup-api
  • pip install yelp

Running the console app

In the linux terminal, type python console.py

Example case of using the console with craigslist

When you enter the console, you will be prompted to choose a database. To see a list of available data sources, type 'options' or 'o'. This time without the quotes. These instructions will be restated within the application. Each data source has can be abbrivated to single letter, so if you type 'c', you get craigslist. Type 'c' now.

Once you have choosen your data source, you have a some commands available for that data source. In this cause, you are going to want to use the command scheduleCollect. The command scheduleCollect schedules the application to scrape between the hours of the start time, and end time. For example, scheduleCollect "23:00" "23:30" would start scrapping at 11am and end at 11:30am. Scrapping would occur at those times each day. The quotes must be included. Pay attention to the clock of the system you are using because it may very. The server time is different then the time on my computer.

Side note: sampleCollect, and sampleStore will be quite useful when testing the collection of data. sampleCollect displays a sample of the data collected from the database, and sampleStore stores an example result to the default database.

Working commands (only craigslist right now)

  • scheduleCollect
  • sampleCollect
  • sampleStore
  • help
  • exit
